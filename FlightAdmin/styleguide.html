Rutsystem

Rutsystemet är indelat i fyra kolumner. Huvudrubriken täcker alla fyra kolumner medan de övriga lådorna täcker två stycken. Dessa lådor är centrerade och anpassas i storlek efter innehållet. Avståndet mellan kolumnerna är lika stort som det mellan raderna, 30 pixlar.

HTML

<div class="grid-container">
<div class="header">Innehåll</div>
<div class="item1">Innehåll</div>
<div class="item2">Innehåll</div>
Här kan fler items läggas in
</div>

CSS

.grid-container {
display: grid;
grid-template-areas:
'header header header header'
'box1 box1 box2 box2'
'box3 box3 box4 box4';
justify-content: center;
grid-gap: 30px;
}

.header { grid-area: header;
text-align: center;
}

.item1 { grid-area: box1;}
.item2 { grid-area: box2; }
.item3 { grid-area: box3; }
.item4 { grid-area: box4;
max-width: 100%;
}

.grid-container > div {
padding: 15px;
padding-top: 25px;
background: rgba(0,0,0,.5);
}
Samtlig text förutom knapptexten har har vit färg.
Till huvudrubriker, underrubriker och knapptexten används typsnittet avenir,
till huvudrubrikerna används dock en tunnare version.
Till formulärrubriker och paragrafer används typsnittet Times-Roman.

På sidan används två färger. Lådorna använder sig av en svart färg med en opacitet på 50%. All text använder sig av vit färg förutom knapptexten som är svart. När man för muspekaren över en knapp så får knappen en grå färg.
#000000
#ffffff
#cccccc
CSS

Färg på lådorna:

.grid-container > div {
background: rgba(0,0,0,.5);
}


Färg på texten:

h1, h2, td, th { color: white; }


Färg på knappar vid hovring:

#skicka:hover {
background: #cccccc;
text-decoration: none;
cursor:pointer;
}

Färg på knapptexten:

#skicka {
color: #000000;
}
Formuläret är centrerat inom lådan. Bredden på textfältet är hälften så brett som lådan det är i. Knappen har rundade hörn och har en vit färg och knapptexten har en svart färg.

I exemplet bredvid används en grå bakgrundsfärg för att förtydliga den vita rubriken.
